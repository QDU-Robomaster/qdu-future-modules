# 青岛大学未来战队电控/视觉软件仓库索引

## 使用方式

### 添加源

```bash
xrobot_src_man add-source https://qdu-robomaster.github.io/qdu-future-modules/index.yaml
[SUCCESS] Added source: https://qdu-robomaster.github.io/qdu-future-modules/index.yaml to Modules\sources.yaml
```

### 列出软件包

```bash
xrobot_src_man list
Available modules:

......
```
